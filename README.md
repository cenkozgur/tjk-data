# TJK Data Mirror

Public mirror of Turkish horse racing data (TJK — Türkiye Jokey Kulübü):
- `data/results/` — Daily race results (JSON per date)
- `data/odds-snapshots/` — Intra-day odds movement (JSONL per date)
- `memory/` — Aggregated stats (horses, jockeys, trainers, tracks) + prediction outcomes log

Data is published by TJK and refreshed automatically via GitHub Actions.
